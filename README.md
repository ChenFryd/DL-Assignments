# Deep Learning Assignments

Course assignments for Deep Learning at Ben-Gurion University of the Negev.

## Assignment 1 — Neural Network from Scratch

Build a fully-connected neural network from scratch using NumPy, without any deep learning frameworks. The goal is to gain a deep understanding of forward and backward propagation.

**Sections:**
1. **Forward Propagation** — parameter initialization, linear forward, ReLU/Softmax activations, batch normalization, and cost computation
2. **Backward Propagation** — gradients for linear, ReLU, and Softmax layers, and parameter updates
3. **Full Training Loop** — L-layer model training with mini-batches and a predict function
4. **Experiments** — hyperparameter tuning and analysis
5. **Conclusions**

Notebook: [Assignment1/neural_network_from_scratch.ipynb](Assignment1/neural_network_from_scratch.ipynb)

## Assignment 2 — Face Verification with Siamese / Metric-Learning Networks

Design and train face verification systems based on Siamese / metric-learning principles on the aligned LFW (LFW-a) dataset. Built in PyTorch. Starting point: Koch, Zemel & Salakhutdinov (2015), *Siamese Neural Networks for One-shot Image Recognition*. The focus is on running fair, well-controlled empirical comparisons across loss functions and backbones, not on chasing a single best number.

**Required experiments:**
1. **Loss function** (backbone fixed to a Koch-style CNN) — Koch BCE on L1, contrastive on L2 (Hadsell et al. 2006), triplet with semi-hard mining (Schroff et al. 2015), plus a random-triplet ablation.
2. **Backbone** (loss fixed to the winner of Experiment 1) — slimmed Koch CNN vs. ResNet-18 trained from scratch, with parameter counts matched within ±20% so the comparison isolates architecture, not capacity.
3. **Frozen pretrained-features baseline** — ImageNet-pretrained ResNet-18 used as a fixed feature extractor with cosine similarity (no training, no fine-tuning).

**Evaluation protocol** (same test pairs and one-shot episodes for every model): verification accuracy at a validation-tuned threshold, ROC + AUC on a single overlay figure, N-way one-shot accuracy for `N ∈ {2, 5, 20}`, training/validation loss curves and wall-clock time. The best model in each experiment is reported as mean ± std across 3 seeds.

Notebook: [Assignment2/face_verification_siamese.ipynb](Assignment2/face_verification_siamese.ipynb) — generated by [Assignment2/_build_notebook.py](Assignment2/_build_notebook.py). Regenerate with:

```bash
python Assignment2/_build_notebook.py
```

Dataset: download the aligned LFW (LFW-a) crops from <https://talhassner.github.io/home/projects/lfwa/index.html> and extract into `Assignment2/lfw2/`. Train/test pairs are in [Assignment2/pairsDevTrain.txt](Assignment2/pairsDevTrain.txt) and [Assignment2/pairsDevTest.txt](Assignment2/pairsDevTest.txt).

# How to Install
1. Have Python > 3.9 and Git
```bash
sudo apt install python3.12-venv -y
```

2. Clone the Repository
```bash
git clone https://github.com/ChenFryd/DL-Assignments.git
cd ./DL-Assignments
```

3. Install the Virtual Environment
```bash
python3 -m venv .venv
```

4. Activate the Virtual Environment

Linux/macOS:
```bash
source .venv/bin/activate
```

Windows:
```bash
.venv\Scripts\activate
```

5. Install the Requirements

For Assignment 1:
```bash
pip install -r requirements.txt
```

For Assignment 2 (PyTorch + extras):
```bash
pip install -r Assignment2/requirements.txt
```
